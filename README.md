# BugTracker
Offline-first Android Bug Tracker App using Kotlin, Room, Retrofit, MVVM, and Git
Bug Tracker


Features

- Create Issue
- Read Issues
- Update Issue
- Delete Issue
- Offline storage with Room Database
- Remote synchronization with Retrofit
- Automatic retry for failed synchronization
- MVVM Architecture
- Git branching workflow
- Version tagging

Technologies

- Kotlin
- Jetpack Compose
- Room Database
- Retrofit
- Git & GitHub

🚧 Under Development
